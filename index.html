Guia Rapida de Comandos Git

git init

Inicializa un nuevo repositorio Git local en el directorio actual.

git status

Muestra el estado actual del repositorio, archivos modificados y no rastreados.

git add .

IAgrega todos los archivos al area de preparacion (staging).

git commit -m "mensaje"

Guarda los cambios agregados con un mensaje descriptivo.

git push

Envia los commits locales al repositorio remoto.

git pull

Ontiene y fusiona los cambios del repositorio remoto.
